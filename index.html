About Christine

"So how did you get into coding?" Glad you asked! When the COVID-19 stay-at-home orders came into effect, I found myself with a lot of extra time on my hands. After spending much of that time learning new cooking recipes/techniques, watching YouTube videos about outer space, and exploring the Palos Verdes Peninsula, I decided to learn about web development (thanks to my husband, Kevin, who is also in the field).

I began following video tutorials and crash courses on YouTube. When I discovered a bug or struggled to understand a programming concept, I reached out to my "teammate"/partner-in-crime to help me debug the code or understand a concept. Working together was in a way, my first experience working as a team of developers. We would both hit a roadblock, discuss, test, debug, research, try again (and repeat this process x number of times) until we finally understood what was going on under the hood and celebrate!

This productive struggle is what motivates me to become a better software developer. It's what gets me excited to work with a team of great programmers in the future as well. One of my goals as a developer is to help create a service or product that will benefit people on a large scale. And as a student at Thinkful, I've officially begun my journey to do just that.

Christine's Projects

L U M I N A

Project purpose & audience: This app enables anyone to see the Astronomy Picture of the Day (APOD) and learn more about outer space by searching for terms in the NASA image and video library. Users may simply search a term of interest and scroll through all that there is to learn about our fascinating universe! The purpose of this project was to practice implementing asynchronous code by fetching data from two different NASA API endpoints.

How it was built: This project was built using HTML, CSS, and the jQuery library in JavaScript. I used jQuery objects to traverse the DOM and DOM manipulation methods to render results based on the user's input. I used the fetch API and the .json() method to grab properties in the JSON object to display content pertinent to the user's search. In the process of building this app, I gained a deeper understanding of how the fetch process works, as well as the importance of understanding how to retrieve specific values based on the JSON response.

Technologies used: HTML, CSS, JavaScript, jQuery, NASA APOD API, NASA Image and Video Library API

Word Beater App

Project purpose & audience: Given a random word and time limit, the user must accurately type the word in the time given to earn points. If the user is unable to type the exact word in the given time, the score is reset to zero and the user can try again by typing the next provided word. This app is for anyone who would like to play a speed typing game to practice typing accurately.

How it was built: This project was built using the setInterval method, callback functions, if/else statements, an array of strings, and Math methods to generate a random word. I also added two new features: a drop-down menu to select the level and a "Try Again" button to reset the game. These extra features were added using query selectors, event listeners, and DOM manipulation. In the process of building this app, I gained a deeper understanding of how conditional statements and HTML DOM manipulation can work together to render a dynamic user experience. Overall, I learned how JavaScript can bring greater interactivity to a website as opposed to just HTML and CSS.

Technologies used: HTML, CSS (Bootstrap), JavaScript

GitHub Repository: Word Beater

Elegant Blur Landing Page

Project purpose & audience: The purpose of this mini project is to create a responsive, elegant opening web page with a blurring effect. When the page is loaded, the background image is blurred and the opening text is clearly shown. When the user hovers over the text, the background image is clarified and the text is blurred. This feature is for anyone who would like to create a visually appealing landing page for their website using this effect.

How it was built: This project was built using the grid system, media queries, transitions, and pseudo-elements. I also added a personal touch by calling it "Christine's Cooking Corner," a fictional website to take courses in cooking. In the process of building this app, I learned more about CSS dynamic properties and responsive styling techiques using media queries.

Technologies used: HTML, CSS

GitHub Repository: Elegant Blur Landing Page
